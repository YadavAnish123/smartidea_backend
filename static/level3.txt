🔴 Level 3: Recognition with Text Prompts
Goal: Test and reinforce identification skills with basic reading.


How it works:


A word is shown (e.g., “Apple / सेब”) and the child must select the correct image from a group.


Immediate audio and visual feedback on correct/incorrect choices.


Encouragement for retry if wrong.


✅ Key Skills Targeted: Early reading, word-image matching, memory development.
